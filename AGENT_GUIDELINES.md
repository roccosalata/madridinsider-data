# Madrid Insider Data Guidelines

## Purpose

This repository is the canonical content database for Madrid Insider.

All content should ultimately originate here before appearing on the production site.

## Legacy Migration Workflow

When importing content from historical Madrid Insider snapshots:

1. Locate the original URL.
2. Verify the content or entity still exists in 2026.
3. Classify the content.
4. Assign it to one of the five categories.
5. Add official links when available.

## Content Status

Every record should contain one of the following statuses:

- Evergreen
- Annual Review
- Dynamic
- Legacy
- Archived

## Five Categories

- Madrid Essentials
- Living in Madrid
- See in Madrid
- Do in Madrid
- Madrid Now

## AI Agent Rules

- Prefer official Madrid, Community of Madrid, and Spanish government sources.
- Preserve useful information from historical versions of Madrid Insider.
- Never duplicate records.
- English-speaking services should be prioritized.
- Madrid-specific content takes precedence over Spain-wide content.
- Favor structured JSON over prose.
